## Funções de Arrays

### array_push($array, 'item') = Insere item no fim do array
### array_shift($array) = Remove item do início do array
### array_unshift($array, 'Item') = Insere item no início do array
### array_pop($array) = Remove item do fim do array
### ksort($array) = Ordena o array pelo índice
### count($array) = Exibe o tamanho do array
### in_array('item', $array) = Verifica se o item passado como parâmetro está no array
### array_merge($array1, $array2) = Une dois arrays em um só
### array_count_values($array) = Exibe a quantidade de vezes que cada item aparece no array
### array_unique($array) = Remove itens duplicados do array
